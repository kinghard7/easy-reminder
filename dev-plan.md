# EasyReminder 催账助手 - 开发计划书

## 1. 项目愿景
**核心价值**：解决“熟人借钱不好意思开口催”的社交痛点。
通过数据化的账单管理和标准化的提醒模版，将原本尴尬的个人催收过程转变为一种正式、礼貌且专业的流程。

## 2. 重新定义业务逻辑
*   **安全准入**：通过超级密码/安全验证进入，保护个人借贷私密数据。
*   **账单生命周期管理**：
    *   **录入**：记录借款人、本金、借款日期及年化利率。
    *   **计算**：系统根据当前日期和年化利率，自动动态计算截止目前的待还总额。
    *   **还款**：支持分批次部分还款，实时更新债权状态。
*   **智能催单引擎**：
    *   **模版化**：提供不同语气（客气、正式、紧急）的预置模版，并支持用户完全自定义。
    *   **动态渲染**：自动填充变量（如：`{name}`、`{amount}`、`{date}`），确保催款信息的准确性与专业度。
    *   **原生触达**：一键调用系统原生短信 (SMS) 或邮件 (Email) 接口，直接跳转至发送界面。

## 3. 技术方案全景
*   **底层框架**: Vite + Vue 3 (Composition API)
*   **视觉规范**: 
    *   **设计风格**: 深度定制的 Dark Glassmorphism（深色玻璃拟态）。
    *   **交互动画**: 基于 Vue Transition 的流畅切换，辅以抖动反馈（Shake Feedback）提示异常。
*   **状态与存储**:
    *   **核心状态**: Pinia 驱动的全局存储（Bills, Templates, Auth）。
    *   **持久化**: 本地加密存储（LocalStorage / Capacitor Storage）。
*   **移动端底座**: Capacitor (支持 Android 原生打包，适配安全区域)。
